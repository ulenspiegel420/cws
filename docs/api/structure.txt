Структуры данных API application/json

Формат ответа 200:
    {
        'success': <True или False>,
        'data': <Данные ответа>
    }

Формат ответа 404:
    {
        'data':
            {
                'resource': <ресурс который не удалось найти>
            }
    }



1. Маршруты модуля basestation:
    1.1 /api/bs/<code> GET
        Данные:
        {
            'id':    <ID записи>,
            'ipaddr': <IP адрес станции>,
            'is_active': <Состояние впн соединения>,
            'receiver_updated_pwd': <Состояние обновления пароля на приемнике>,
            'problems': [<Список проблем>]
        }

    1.2 /api/bs GET
        Данные:
        {
            <Код БС>: {
                'id':    <ID записи>,
                'ipaddr': <IP адрес станции>,
                'is_active': <Состояние впн соединения>,
                'receiver_updated_pwd': <Состояние обновления пароля на приемнике>,
                'problems': [<Список проблем>]
            }, ...

        }

    1.3 /bs/ipaddr GET
        Данные:
        {
            <Код БС>: <IP адрес станции>, ...
        }
